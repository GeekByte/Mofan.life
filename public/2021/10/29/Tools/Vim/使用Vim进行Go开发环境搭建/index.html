使用Vim进行Go开发环境搭建

Last updated on 2 months ago

本文档假定你已经搭建好了Go开发环境，重点介绍如何配置Vim.

一般 Mac OS 或者 Linux 都内置了Vim, 所以，下面的插件都要用其安装方式描述中的 vim-plugin 的方式安装, 不过在此之前，需要首先为 Vim 安装 vim-plugin，安装方式请参考fatih/vim-go: Go development plugin for Vim.

安装完 vim-plugin 后，可以使用下面各个插件提到的 vim-plugin 方式进行安装:

  • fatih/vim-go: Go development plugin for Vim
  • neoclide/coc.nvim: Nodejs extension host for vim & neovim, load extensions like VSCode and host language servers. 注意：如果已经安装过nodejs，且符合其要求，则不必再执行其安装nodejs的步骤.

安装完coc vim 后，可以去安装他的extensions（Using coc extensions · neoclide/coc.nvim Wiki），找到coc-go进行安装，如果需要安装其他的拓展，也可以进行安装.

安装完上面连个必须插件后，可根据需要尝试其他的插件，比如:在左侧展示目录树的 NERDTree.

下面是我的一份 .vimrc 的配置文档，可进行参考，当然你也可以直接替换你的 .vimrc 文档，然后在文档里执行 :PlugInstall 的 vim 命令(注意一定要先安装 vim-plugin).
