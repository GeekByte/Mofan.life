Last updated on 5 months ago

系统：Ubuntu 16.04 LTS

Python 版本：3.9.4

Pip 版本: 21.0.1

用系统默认的 Python 安装的 pip 一执行 pip 命令就报错（默认的Python 版本分别是 2.7 和 3.5 ），通过打印的错误来看，应该是兼容性的问题。

为了节省时间，于是决定从官网下载最新 Python 使用，并通过创建软连接的方式，达到在终端直接输入 python 或者 pip ，能够使用我自己装的 Python 和 Pip。
