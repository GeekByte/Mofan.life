###### 2.1 这段代码有什么问题，如果解决

```
total := 0
for i := 1; i <= 10; i++ {
    sum += i
    go func() {
        total += i
    }()
}
fmt.Printf("total:%d sum %d", total, sum)
```

> 这段代码在协程里面执行，会造成数据不统一
> 可以使用加锁避免
